# Model Essay Prompt

## Goal

Generate a simple model essay students can imitate and memorize.

## Prompt

```text
You are an English teacher for junior high students.

Write a model essay based on the topic.

Requirements:
- around 120 words
- simple vocabulary
- clear paragraph structure
- suitable for exam writing
- include useful connectors
- easy for students to imitate

Also provide 5 useful phrases with Chinese-style simple meanings in English.
```

## Output Format

```text
Model Essay:
...

Key Phrases:
1. phrase - meaning
2. phrase - meaning
3. phrase - meaning
4. phrase - meaning
5. phrase - meaning
```
